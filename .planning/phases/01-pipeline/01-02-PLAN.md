  <action>
Create `detector.py` implementing `run_detector(from_streamer: Queue, to_viewer: Queue)`.

The detector uses a stateful approach matching the basic_vmd.py algorithm (frame differencing against previous frame). Adapt the algorithm from `basic_vmd.py` as an inline function — do NOT import basic_vmd as a module (it has no importable functions, only a `__main__` block).

Algorithm (from basic_vmd.py, adapted):
- Maintain `prev_gray = None` across loop iterations.
- For each frame:
  1. Convert BGR frame to grayscale: `gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)`
  2. Apply Gaussian blur before subtraction (noise reduction per user decision): `gray = cv2.GaussianBlur(gray, (21, 21), 0)`
  3. If `prev_gray` is None, set `prev_gray = gray` and continue (no detection on first frame).
  4. Compute diff: `diff = cv2.absdiff(gray, prev_gray)`
  5. Threshold: `thresh = cv2.threshold(diff, 25, 255, cv2.THRESH_BINARY)[1]`
  6. Dilate: `thresh = cv2.dilate(thresh, None, iterations=2)`
  7. Find contours: `cnts, _ = cv2.findContours(thresh.copy(), cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)` (use `cv2.findContours` directly; `imutils.grab_contours` not needed with this call signature)
  8. Filter: keep only contours where `cv2.contourArea(c) >= MIN_CONTOUR_AREA`. Use `MIN_CONTOUR_AREA = 500` (discretion per CONTEXT.md).
  9. Update `prev_gray = gray`.
  10. Contours are numpy arrays — they ARE picklable for Queue.

Loop body `run_detector`:
1. `msg = from_streamer.get()` — blocks until message available.
2. If `msg is EOS_SENTINEL`: put `EOS_SENTINEL` on `to_viewer` and return.
3. Attach to SharedMemory: `shm = SharedMemory(name=msg.shm_name, create=False)`.
4. Reconstruct frame: `frame = np.ndarray(msg.frame_shape, dtype=msg.frame_dtype, buffer=shm.buf)`. Make a copy: `frame = frame.copy()` — this is needed before closing shm so Detector's numpy array is independent of the memory block.
5. Close (not unlink) local shm handle: `shm.close()`.
6. Run VMD on the frame copy to get filtered contours.
7. Build `DetectorMessage(shm_name=msg.shm_name, frame_shape=msg.frame_shape, frame_dtype=msg.frame_dtype, frame_index=msg.frame_index, fps=msg.fps, contours=list(cnts))`.
8. Put on `to_viewer`.

Note: Detector closes its shm handle after copying the frame. The original SharedMemory block is NOT unlinked here — Viewer still needs to access it (via `msg.shm_name`). This matches the lifecycle comment in streamer.py: Viewer is responsible for unlink.

Wait — reconsider: Viewer needs to read the raw frame from SharedMemory too (to display annotated version). So we must NOT copy and discard in Detector. Instead:
- Detector reads frame to run VMD (copy for its own numpy use), then puts the original shm_name in DetectorMessage so Viewer can attach to the same block.
- Detector closes its local handle (`shm.close()`), does NOT unlink.
- Viewer attaches, reads, draws, displays, then unlinks the block.

Imports: `cv2`, `numpy as np`, `multiprocessing.shared_memory.SharedMemory`, `multiprocessing.Queue`, `from ipc import ShmFrameMessage, DetectorMessage, EOS_SENTINEL`.
