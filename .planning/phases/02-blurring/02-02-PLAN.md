From C:/code/VideoStreamAnalysis, stage only viewer.py and create a commit explaining the design decision:

```
git add viewer.py
git commit -m "feat(phase-b): blur motion regions in Viewer before drawing bounding boxes

Applies cv2.GaussianBlur to the full bounding rectangle of each detected
contour in viewer.py (_blur_motion_regions). Chose full bounding-rect blur
(rather than precise contour polygon mask) because: it satisfies BLUR-02
exactly, avoids the complexity of polygon mask creation, and produces a
clean visual result where the extent of each motion region is clearly
indicated by the blur boundary. Kernel size scales proportionally with the
bounding box (BLUR_KERNEL_FRACTION = 0.2 of the smaller dimension), clamped
to odd values in [3, 99], so small and large regions each receive
perceptually appropriate blur strength."
```
